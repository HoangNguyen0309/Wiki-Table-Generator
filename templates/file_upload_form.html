Excel to MediaWiki

Upload a text file in MediaWiki format to turn into an excel

Upload a an excel file to turn into WikiMedia format

If you uploaded the wrong file format, the webpage will throw an error.

Upload excel

Excel file will be turned into a .txt file in single mark "|" MediaWiki format.

Excel file will be turned into a .txt file in double mark "||" MediaWiki format.

Upload text

Text file containing the table in MediaWiki format will be turned into an excel. (Currently only working with single mark table)
